# Codex Web Manager

面向移动办公的本地 Web 控制台，用于通过手机或浏览器远程查看和控制本机 Codex 会话，并统一管理多个账号、API 和 Profile。

## 下载与安装

公开发布包、版本说明和历史版本统一放在：

**[Townhy/codexmanager-releases](https://github.com/Townhy/codexmanager-releases/releases)**

下载最新的 `codex-web-manager-<version>-win-x64.zip`，解压后运行 `Start Codex Web Manager.cmd`。

运行环境：Windows 10/11、Node.js 22.5+、Codex Desktop。

此仓库仅作为项目介绍和下载入口，不包含源码、账号信息、API Key 或本机会话数据。
